# DateStructure, Please

A data-structure-focused matchmaking moderator game prototype.

Implemented structures:

- Directed graph for forbidden relationship paths
- General tree with LCA distance for hobby compatibility
- Weighted graph with Dijkstra shortest paths for long-distance penalties
- Stack for UI overlays
- Queue for staged game events
- Merge sort and binary search helpers for profile management

Run the console demo:

```powershell
python main.py
```
